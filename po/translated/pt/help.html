Ajuda do USB Creator

Descrição

O USB Creator é uma ferramenta para ajudar os utilizadores, a criar uma pen drive de arranque com vários sistemas operacionais Linux.

Importante

Certifique-se de que você fez o backup de todos os dados importantes da pen drive antes de começar.

Comandos

Você também pode usar o Criador USB a partir do terminal. Aqui estão alguns exemplos:

# usb-creator -f -i /path/to/my.iso -m

Isto irá formatar a unidade flash, copie o ficheiro ISO para a unidade flash e verifique o hash do ficheiro ISO copiado com o ficheiro ISO original.

# usb-creator -b -g -r

Isto irá tentar reparar uma unidade flash com arranque que já contenha arquivos ISO.

# usb-creator -h

Isto irá mostrar todas as opções disponíveis e seu uso.

Características ocultas

Quando você remove o nome do arquivo ISO a partir do campo ISO, USB Creator irá calcular o espaço necessário para todos os arquivos ISO nesse diretório.

Você pode alterar a ordem na qual os arquivos ISO são exibidos no menu, adicionando um número na frente do nome do arquivo ISO, por exemplo: "01-MyDistro-amd64.iso".

Os links simbólicos são seguidos, mas os links simbólicos devem ter a ".iso" extensão para que ele funcione.

A ISO não está a iniciar?

No nosso página GitHub você pode ver uma lista de distribuições testadas, mas nem todas podem ser iniciadas diretamente a partir do arquivo ISO. Verifique no fórum da distribuição, se ela pode ser iniciada diretamente a partir do ficheiro ISO.

O menu está configurado num ficheiro que você pode encontrar na sua pen drive: /boot/grub/grub.cfg.

Por favor, ajudem a melhorar esta aplicação, adaptando grub.cfg até o arranque da ISO, e publicar as alterações no GitHub ou no nosso forum.

Não se esqueça de anexar o ficheiro log quando você postar um problema no nosso fórum: /var/log/usb-creator.log
